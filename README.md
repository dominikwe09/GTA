# GTA (Java + GLOOP)

A small GTA-inspired school project built in Java using the GLOOP library.

## Features

- Run around a 3D world
- GTA-style gameplay concepts (shooting :D)
- Custom textures and assets
- HUD (crosshair + ammo)
- Enemies with Health Bar

## Project Structure

```

src/        # Java source code
lib/        # Required libraries
Texturen/   # Textures and game assets
doc/        # Documentation GLOOP etc

````

## Getting Started

### Requirements

- Java JDK
- GLOOP library

### Run

Use the provided startup script:

```bash
start.bat
````

### Build

```bash
build.bat
```

## About

This project was created as a school project to explore game development with Java and GLOOP. It is inspired by the Grand Theft Auto series and is intended for educational purposes only.
